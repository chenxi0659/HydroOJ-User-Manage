# HydroOJ插件 用户管理面板

一个为 HydroOJ 提供可视化用户管理功能的插件，允许管理员在控制面板中方便地管理用户信息、权限和状态。

## 安装方法

```bash
sudo su
cd /root/.hydro/
git clone https://github.com/SummerofOrange/hydrooj-user-management
hydrooj addon add /root/.hydro/hydrooj-user-management
pm2 restart hydrooj
```

## 使用方法

1. **访问用户管理**: 登录 HydroOJ 后，在控制面板侧边栏找到"用户管理"菜单项
2. **搜索用户**: 在用户列表页面使用搜索框查找特定用户
3. **编辑用户**: 点击用户列表中的"编辑"按钮进入用户详情页面
4. **管理权限**: 在用户详情页面的"权限管理"部分设置用户权限
5. **重置密码**: 在"密码管理"部分为用户重置新密码
6. **封禁用户**: 在"用户状态"部分封禁或解封用户
7. **删除用户**: 在用户列表勾选用户后点击“删除选中”，确认警告并输入当前管理员密码；该操作会删除账户、所有域成员资格及用户组引用，但保留历史提交记录。当前管理员、UID 1 与 root 账户受到保护，无法删除。
8. **批量清空专业/班级**: 在用户管理列表勾选用户后，可分别点击“清空选中专业”或“清空选中班级”；确认后会自动同步专业、班级分组，且不会改变注册方式。
9. **宽屏用户管理布局**: 在安装 Hydro Front Beauty 全局布局插件后，用户管理与其他 Hydro 页面都会保留原右侧选项卡；标准双栏页主内容区最大约 1152px（上一版约 1440px 的 0.8 倍），右侧栏保持约 300px。用户管理的页面操作位于左侧、批量操作位于右侧，空间不足时整组换行，所有按钮均留在独立操作区内；分组筛选操作左对齐。

## 权限说明

插件使用以下权限级别：

- **-1**: root（超级管理员）
- **0**: 已封禁用户
- **4**: 系统保留
- **8**: 访客用户
- **16842756**: 默认用户权限
- **其他值**: 自定义权限

## 安全特性

- ✅ 权限验证：只有具有系统管理权限的用户才能访问
- ✅ 操作确认：重要操作（如重置密码、封禁用户）需要确认
- ✅ 权限保护：防止非超级管理员修改超级管理员账户
- ✅ 数据验证：自动验证用户名和邮箱的唯一性
- ✅ 输入验证：前端和后端双重验证用户输入
- ✅ 删除二次确认：删除用户必须先确认风险，再输入当前管理员密码，服务端再次验证密码
